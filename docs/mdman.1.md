# NAME
mdman - man pages in markdown

# SYNOPSIS
**mdman** [_OPTIONS_] _FILE_

# DESCRIPTION
**mdman** allows for generation of man pages from a subset of Github flavored markdown.

# OPTIONS
**-h**, **--help**
    Print help message.

**-V**, **--version**
    Print version information.
